Prerequisite :
1. Python (version 3)
2. Requests Module
	Download it using :
    	pip install requests

Install the package using :
pip install src/dist/simulation-0.1-py3-none-any.whl

Api usage :
simulation filename 
where filename is the sensor data file that you want to run simulations against.

Response :
{	
	message : "success",
	images : [
	]
}
